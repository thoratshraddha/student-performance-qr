 
# Student Performance Tracking System Using QR Code

A web-based tracking system for monitoring student performance using QR code technology, built during internship at Wolfox Services Pvt. Ltd.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** Python (CGI)
- **Database:** MySQL
- **Other:** QR Code Integration

## Features
- 📱 QR code scanning for student identification
- 📊 Role-based dashboards (Admin, Teacher, Student)
- 📝 Student performance monitoring and tracking
- 🔐 Authentication and secure login
- 🗄️ MySQL database integration
- 📈 Performance reports and analytics


## How to Run
1. Clone the repository
2. Install XAMPP or any Apache server
3. Place project folder in `htdocs` directory
4. Setup MySQL database
5. Open browser and run via Apache server
